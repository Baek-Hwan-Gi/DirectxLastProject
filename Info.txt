Implement an interactive application of 3D graphics by using DirectX 11 SDK (with MS Visual Studio 2015).
? Present your project plan.
- Form your group up to 2 people.
- Use MS PowerPoint slides (more than 10 pages) and present the following:
? Title, motivation, expected main scene (picture), UIs, controls, resources, role allocation.
? Include at least 3 characteristics (differences from others) of this project.
? Implement the following requirements:
- Build your project upon the framework (tutorial) codes used in the class.
- Create the 3D world.
? All objects should be placed in the same world space.
? Use more than 10 different 3D objects (meshes) loaded from .obj files.
? All objects should be properly textured.
? Use a background from an image or a 3D mesh.
- Navigate the world.
? Use camera translations and rotations in order to navigate through the world.
? Use both keyboard and mouse devices.
? Maintain at least 30 fps all the time.
- Lighten the world.
? Use ambient, diffuse, and specular lightings.
? Use more than 2 different lighting source types: point, directional, spot, etc.
- Attach user interfaces (UIs).
? Show rendering information: FPS, CPU, objects, polygons, screen size, and one more data (useful).
? Show texts clearly in your world by using font images.
? Attach more than 2 different UIs to help your navigation through the world.
- Animate the objects.
? Add at least 3 objects should move (translate and rotate) in the world (e.g. NPCs).
? All moving objects should translate and rotate differently.
- Show visual effects.
? Show at least 3 different special effects (something different from tutorials examples).
- Write an implementation report.
? Use a .docx or .hwp file (more than 2 pages).
? Describe application functions, controls, implementation issues, any difficulty, role allocation, etc.
? Evaluation
- Plan: 20%, Mid-check: 10%, Implementation: 70% (Requirements: 40%, Quality: 30%)
? Submission
- Compress all files (source codes and resources) into a .ZIP file and submit to ClassNet.

? Precautions
- A project copied from others or a previous work will be scored 0 (both provider and copier).
- A project NOT opened and buildable from DirectX 11 and MS Visual Studio 2015 will be scored 0.
- A project NOT presented or submitted after the due date will be scored 0.
